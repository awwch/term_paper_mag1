Cоставляется список стоп-слов, выделяемых модулем NLTK, куда входят служебные слова, местоимения, некоторые наречия, все формы глагола «быть». 
При дальнейшей работе программы данный список пополняется.
На чтение подаётся файл со словарём, представленным в виде таблицы со столбцами «слово» и «определение», строка которой соответствует словарной статье. 
Перед первичной обработкой, включающей очистку от небуквенных символов, токенизацию и лемматизацию, столбцы объединяются. 
К получившейся структуре применяются функции, обеспечивающие препроцессинг.
Функция dict_to_wordlist отвечает за очистку текста от ненужных символов, осуществляемую с помощью регулярных выражений, 
приведение его к нижнему регистру, и получение грамматической информации о слове на базе библиотеки pymorphy2, и возвращает двухуровневый список,
содержащий словари, где слову в той форме, в которой оно представлено изначально, соответствует лемма, и где лемме присваивается значение-грамматический тег.
Функция dict_to_sentences осуществляет токеницацию на основе библиотеки NLTK, использует описанную выше dict_to_wordlist и отвечает за сохранение текста словаря в виде предложений, соответствующих толкованиям. 
В ходе работы функции поданный на вход текст преобразуется в структуру, представляющую его в виде списка предложений, каждое из которых является двухуровневым списком c метаинформацией о содержащихся в нём словах. 
При этом из толкований удаляются стоп-слова, список которых взят из библиотеки NLTK, он также был рсширен при подсчёте наиболее частотных лемм, 
который выявил отсутствующие в первичном списке вспомогательные части речи, местоимения, а также некоторые отдельно представленные морфемы, которым в словаре даётся толкование.
В результате работы описанной части программы, помимо представленной структуры всего текста словаря, создаются списки всех лемм, существительных с их частотностью, а также список лемм с указанием количества словарных статей, в которых они встречаются.
Оставшаяся часть программы производит рассчет меры Дайса для выбранных ключевых слов по всему словарю, равной отношению количества совпадающих слов в двух дефинициях к сумме слов в его толковании по всему словарю.
